Widgets showcase

This App made with Angular showcases a few reusable frontend components.

You can use the links above to visit the various routes of the application, each one corresponds to a different module and showcases a few components.

The components are based on those available in Semantic UI, with a few extra customizations. For this reason, the app uses the CSS part of Semantic UI for styling, without using the JS part of the library.

These are some of the Angular functionalities used in this app:

  • Router
  • Modules
  • Lazy loading
  • Directives (built-in and custom)
